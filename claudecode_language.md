要将 Claude Code 设置为中文，你可以通过交互式命令或修改配置文件来实现。
1. 使用交互式命令设置（推荐）
   在 Claude Code 的对话界面（REPL）中，你可以直接通过内置指令更改语言：
   输入 /config：这会打开一个交互式的设置面板。
   选择 Language：在设置面板中找到语言选项，输入“简体中文”或“Chinese”并保存。
   直接指令：你也可以在对话框中输入 # 后跟指令，例如：# 以后请使用中文回复我。这会将其保存到 User Memory（用户记忆）中，后续所有对话都会自动转为中文。
   太平洋科技
   太平洋科技
   +3
2. 修改全局配置文件
   你可以通过编辑本地配置文件来永久锁定中文输出。
   文件路径：~/.claude/CLAUDE.md（如果文件不存在，请手动创建）。
   添加内容：

```
## 语言规范
- 所有对话和文档都使用中文。
- 尽可能使用中文解释代码逻辑。
```


Claude Code 会在启动时自动读取此文件并遵循其中的指令
