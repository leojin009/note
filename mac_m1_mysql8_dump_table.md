
### mysql从a库a表导入到b库b表

第一步：开启导入向导
在 DBeaver 中，展开 B 连接（目标库），找到 B 表。
右键点击 B 表 -> 选择 导入数据 (Import Data)。

第二步：配置数据源
选择 数据库表 (Database table)，点击 下一步 (Next)。
在“源 (Source)”选择界面，点击右侧按钮选择 A 连接（源库）下的 A 表，点击 下一步。

第三步：精确映射字段（解决 A/B 字段到 C/D 字段）
在 映射 (Mapping) 界面，你会看到一个左右对照的表格：
左侧 (Source Column)：源 A 表的字段。
右侧 (Target Column)：目标 B 表的字段。
操作：点击右侧的单元格，从下拉列表中选择对应的目标字段。
找到 A 字段，右侧下拉选 C 字段。
找到 B 字段，右侧下拉选 D 字段。
忽略其他字段：如果 B 表的其他字段不需要数据，在右侧选择 [none] 或 [skip]。

第四步：调整性能选项
点击 下一步 进入设置页：
Commit size：建议填 500 或 1000（每 1000 条提交一次，速度更快）。
Insert method：选 INSERT。

第五步：获取 SQL 或直接执行
如果你想直接迁移：直接点 完成 (Finish)，DBeaver 会开始搬运数据。
如果你只需要 SQL 文本：
在点击完成前，点击底部的 "SQL 预览 (Script)" 按钮。
DBeaver 会展示根据你的映射关系生成的 INSERT 语句
